# Windows 11 UI Clone 💻 ✨

A fully interactive Windows 11 desktop experience built using pure web technologies.

## 🚀 Features
* **Real-time Taskbar:** Dynamic date and time updates synced with your system.
* **Control Center:** Fully functional brightness and volume sliders with interactive progress bars.
* **Interactive Start Menu:** Smooth toggle animations for the central Windows menu.
* **Glassmorphism UI:** High-fidelity CSS effects including `backdrop-filter` for that authentic Windows 11 feel.
* **Responsive Design:** Optimized for various screen sizes using flexible layouts.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the desktop and taskbar.
* **CSS3:** Advanced styling including Flexbox.
* **JavaScript:** DOM manipulation for real-time logic and UI interactions.
